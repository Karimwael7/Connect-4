



INTRODUCTION:
 Connect 4 is a classic two-player strategy game where the goal is to connect four of your colored discs in a row horizontally, vertically, or diagonally before the second player does. This game is easy to learn and can be played by all ages.
HOW TO INSTALL? 
1/Download the Connect 4 game package from the provided link.
2/Unzip the downloaded file into a directory of your choice.
3/Make sure to download pygame to download pygame follow these steps:
a) Make sure that idle python 3.11 is downloaded
b) Open Command Prompt (Windows) or Terminal (macOS/Linux).
c) Write on the Command Prompt pip install pygame.

HOW TO USE:
Launch the game by running the main executable or script file (connect4.py for Python).
First press a bottom from the 4 bottoms (Play,Rules,Credits,Exit)
PLAY BOTTOM :If you pressed play the game will ask the user to enter his name then the game will ask the user to choose from multiplayer and computer 
RULES BOTTOM :If you pressed Rules a displayed screen of the connect 4 rules will display
Players take turns dropping their colored discs into the columns.
The first player to connect four discs in a row wins the game.
